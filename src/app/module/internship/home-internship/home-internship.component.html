Internship

From the 28th of May to the 3rd of June 2022 I was an Intern at Integration Designers located at Kontich, Antwerp (Belgium).

This company is part of the Cronos group and I8C and exists of twenty consultants.
They work on Integration solutions, based on the IBM technology, which means that they make it possible for different softwares to communicate with each other.
They also design, implement and fulfil both small and big integration projects for their costumers.
They have experts in Datapower, IMB MQ (including the appliance), IBM API Connect etc.
Apart from the implementing of integrations, they provide supporting tasks and technologies.

These days many companies are working on some kind of digital transformation.
They want to offer their services through digital channels.
They want to share valuable data with internal and external parties to get a more effective internal working or create new sources of income.
Data gets offered more and more in the form of APIs.
These days more and more companies choose to use GraphQL API instead of REST API.
GraphQL API is added to IBM API Connect since version 10.

I developed, secured and published a GraphQL API using IBM API Connect and the Apollo Server.
