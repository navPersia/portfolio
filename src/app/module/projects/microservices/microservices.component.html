Shop Microservices

A Webshop API.

Skills:

  • JAVA
  • Spring Boot
  • Docker
  • DevSecOps
  • MongoDB
  • PostgreSQL
An API that connects people to each other. Sellers can put their products on the website, buyers can find these products and then contact the seller. The buyers can leave a rating with comments about the seller later.

This webshop API was developed using microservices architecture and two different databases, one SQL and non SQL.
I designed three subservices and one edge service:
User service: this service is responsible for users' login and keeps users' information.
Product service: this service is responsible for products' information.
Review service: this service is responsible for users' reviews.
